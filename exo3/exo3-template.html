Example of an html poster for Exo3-Online

Sarah Voorbeeld (Universiteit van Amsterdam), Max Mustermann (Heidelberg University), Linda Examplia (Cambridge University)

Abstract
This example shows how you can create with some simple HTML, and powered by the Bootstrap 3 CSS framework, a multimedia poster for Exoplanets 3 Online. You only need to submit the HTML file (which is only a few kilobytes long). All the images and movies are simply linked to a server where you place your material. Movies are easily embedded by putting them on youtube, and using the iframe HTML command. Just look at the HTML source of this poster to see how it works. Note that Bootstrap assures that the poster elements scale properly when viewed on handheld devices. This example is very rudimentary. You can make it much better, e.g improving the layout, adding backgrounds, placing links to your papers, you name it. Use your creativity!
